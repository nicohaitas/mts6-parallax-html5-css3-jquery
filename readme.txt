Quick setup — if you’ve done this kind of thing before

https:
	https://github.com/nicohaitas/mts6-parallax-html5-css3-jquery.git
SSH:
	git@github.com:nicohaitas/mts6-parallax-html5-css3-jquery.git

…or create a new repository on the command line

	echo "# mts6-parallax-html5-css3-jquery" >> README.md
	git init
	git add README.md
	git commit -m "first commit"
	git remote add origin git@github.com:nicohaitas/mts6-parallax-html5-css3-jquery.git
	git push -u origin master

…or push an existing repository from the command line

	git remote add origin git@github.com:nicohaitas/mts6-parallax-html5-css3-jquery.git
	git push -u origin master
